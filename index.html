CSS Layout: The Position Property

CSS Position Properties

Property Description
Static The default position for HTML elements
Relative Positioned relative to its NORMAL position
Fixed Relative to viewport. Stays in the same place when scrolling
Absolute Positioned relative to its nearest ancestor element
Sticky Relative to the users scroll position. No scroll=no sticky

Other Position Properties

Property Description
Bottom Sets the bottom margin edge for a positioned box
Top Sets the top margin edge for a positioned box
Left Sets the left margin edge for a positioned box
Right Sets the right margin edge. Obvs
Clip Clips and absolutely positioned element

What should this all look like?

I find the position selector in CSS particularly confusing and I'm not sure why. But lets put an example together for what all this should look like:

h1 {
  position: fixed;
  top: 50px;
  left: 100px;
}

So what this does is position the h1 element to always be 50px from the top, 100px from the top, relative to the frame/wondow! Magic!
